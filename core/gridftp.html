class iceprod.core.gridftp.GridFTP[source]

Asyncronous GridFTP interface to command line client. Designed to hide the complex stuff and mimic tornado http downloads.

Example

GridFTP.get(‘gsiftp://data.icecube.wisc.edu/file’,
callback=endfunc, streaming_callback=streamfunc)
