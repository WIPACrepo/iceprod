Login¶

Users can log in to IceProd like any other web application.

Authentication is either handled internally, or by an external plugin such as LDAP.

Note

LDAP authentication is set up for IceCube users.

Two Factor Authentication¶

Certain pages and actions require additional authentication. When a new account is registered with IceProd, two-factor authentication must be set up.

IceProd uses time-based one-time passwords (RFC 6238), as used by Google, GitHub, and others. Using a phone app such as Google Authenticator, a barcode image specific to each user can be scanned and registered. The app will then generate short codes valid for 30 seconds each.
