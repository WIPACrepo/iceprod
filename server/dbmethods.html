Handle global queueing request from a site.

For a task to queue on a site, it must be matched in the dataset gridspec list (or the list should be empty to match all), and the necessary resources should be available on the site.

Parameters:
  • resources – (optional) the available resources on the site
  • queueing_factor_priority – (optional) queueing factor for priority
  • queueing_factor_dataset – (optional) queueing factor for dataset id
  • queueing_factor_tasks – (optional) queueing factor for number of tasks
  • num – (optional) number of tasks to queue
Returns:

(via callback) dict of table entries to be merged
